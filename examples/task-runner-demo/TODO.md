## Create an empty placeholder file

Create an empty file named `placeholder.txt` in the current directory.
Do not put anything in it — an empty file is exactly what we want.
Do not change any other files.

## Write a greeting file

Create a file named `hello.txt` containing a single friendly line:

    Hello from the tclaude task runner!

Do not change any other files.

## Run a noop command and capture its output

Run the shell command `echo "noop ok"` and write its exact standard output
to a file named `noop-output.txt` (so the file should contain the line
`noop ok`). This demonstrates running a harmless command inside a task.
Do not change any other files.

## Append a completion marker to the demo log

Append the line `Demo complete.` to a file named `demo-log.md`,
creating the file if it does not exist yet.
Do not change any other files.
