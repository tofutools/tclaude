Visual editor for ~/.tclaude/config.json. It covers the settings this version of tclaude recognizes. Edits stay in this form until you press Save changes — which shows a diff to confirm before anything is written. Most settings apply on next use; a few resolved when agentd starts (spawn rate-limit, clone cooldown) take effect after an agentd restart.

Advanced — sudo configuration (raw JSON)

The agent.sudo block (time-bounded permission elevations, with optional per-conversation overrides) is edited as raw JSON — it is the one nested-map structure without a dedicated form. Leave blank for none. Validated on save.

Edit permanent permissions

PERMANENT — these per-agent overrides persist until you change them again. They are not the time-bounded “+ sudo” elevation: a sudo grant expires on its own, an override here does not. Grant adds a slug, Deny blocks one the global defaults would otherwise give, and Default inherits the global default.
